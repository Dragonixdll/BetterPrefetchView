# BetterPrefetchView

Advanced Prefetch analysis and cheat detection tool for Windows.

## About

This project is based on the original PrefetchView but has been significantly upgraded and modified. The code was not written entirely from scratch - it builds upon existing work with substantial improvements.

## Changes and Improvements

- Yara Detection: Rewrote detection logic, removed generic rules causing false positives, added specific cheat tool detection
- UI Improvements: Modern dark theme with purple/cyan colors, loading animations, better layout
- Code Optimization: Refactored folder structure, improved naming conventions, performance enhancements
- New Features

## Features

- Prefetch file analysis with execution history
- Digital signature verification
- Yara-based threat detection
- Hooking detection
- Advanced filtering and search
- Real-time monitoring

## Requirements

- Windows 10/11 (64-bit)
- Administrator privileges
